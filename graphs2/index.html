graphs2

Norges Bank chart studio

Keep the original graphs/ untouched, and use this workspace to turn CSV, JSON specs, quarterly paths, monthly trackers, or yearly series into exportable Norges Bank-style charts.

1. Start from a preset

The current 2x2 PPR dashboard lives here as the default preset, and there are extra monthly and yearly examples to demonstrate multiple resolutions.

2. Render or tweak the JSON spec

The JSON spec is the nimble fallback when you want full control. CSV import builds a spec for you, then you can fine-tune titles, axis limits, legend labels, plot lines, and series styles here.

Supported chart resolutions: quarter, month, year, date, and numeric.

3. Build a spec from CSV

Paste CSV or TSV, or upload a file. Map the columns once, then graphs2 groups rows into charts and series automatically. The generated spec lands in the editor so you can keep shaping it. 

Expected columns can be as simple as:

chart,series,period,value
KPI,Historisk,2024-Q1,4.0
KPI,Historisk,2024-Q2,3.2
KPI,Anslag,2025-Q1,2.7
KPI,Anslag,2025-Q2,3.0
