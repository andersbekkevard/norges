# Norges Bank Bankplassen - Finansiell stabilitet

Arkiv over blogginnlegg fra [Bankplassen](https://www.norges-bank.no/bankplassen/kategorier/finansiell-stabilitet/), Norges Banks blogg, i kategorien **Finansiell stabilitet**.

Artiklene er skrevet av ansatte i Norges Bank og dekker temaer som boligmarkedet, bankregulering, kryptovaluta, betalingssystemer, klimarisiko, forbruksgjeld og makroøkonomisk stabilitet. De fleste artiklene er på norsk, noen på engelsk.

## Innhold

74 artikler fra 2018 til 2026, organisert etter publiseringsår:

```
articles/
  2018/    6 artikler
  2019/   11 artikler
  2020/   15 artikler
  2021/   11 artikler
  2022/   12 artikler
  2023/    8 artikler
  2024/    1 artikkel
  2025/    8 artikler
  2026/    2 artikler
```

Hver fil er i Markdown-format og inneholder tittel, dato, forfatter(e), lenke til originalen, ingress og fulltekst.

## Oppdatering

Kjør `main.py` for å hente artiklene på nytt:

```
uv run python main.py
```

Krever `requests` og `beautifulsoup4` (installeres med `uv add requests beautifulsoup4`).

## Kilde

Alt innhold er hentet fra https://www.norges-bank.no/bankplassen/ og tilhører Norges Bank.
